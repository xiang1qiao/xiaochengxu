# 小程序改为白色基调简约风格计划

## 设计目标
将当前未来科技风（深色+渐变+毛玻璃）改为白色基调、简约大气的风格

## 颜色方案
- **背景色**：纯白 #FFFFFF + 浅灰 #F5F5F5
- **主色调**：经典蓝 #1677FF 或 黑色 #333333
- **卡片**：纯白 #FFFFFF + 轻微阴影
- **文字**：深灰 #333333 + 中灰 #666666
- **边框**：浅灰 #E8E8E8

## 改造内容

### 1. 全局样式 (app.wxss)
- 背景改为白色 #FFFFFF
- 移除毛玻璃效果和渐变光效
- 使用简洁的阴影代替
- 保持简洁的动画效果

### 2. 首页 (index)
- 移除深色背景和径向渐变
- 卡片改为纯白 + 简约阴影
- 标题使用黑色/深灰色

### 3. 菜单组件
- 卡片改为白色背景
- 图标使用单色或简洁样式
- 保持适当的间距和圆角

### 4. 轮播图
- 简化颜色，使用清爽配色
- 保持圆角和阴影

### 5. 个人中心 (home)
- 白色背景
- 简约的卡片设计
- 清晰的文字层次

### 6. 导航栏
- 白色背景
- 深色文字
- 简洁的选中状态
